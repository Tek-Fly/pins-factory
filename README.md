# Pinterest Factory (Gift Getta) — Private Internal App

Internal tool to automate Pinterest asset creation for Shopify POD mugs:
- Sync products from Shopify
- Generate Pinterest copy packs
- Generate lifestyle mockups via RunComfy
- Build Canva Bulk Create packs (CSV + assets) — no Canva API required
- Ingest Canva exports and output Pinterest/Tailwind-ready CSVs

## Stack
- Next.js (App Router) + TypeScript
- Clerk Auth
- Prisma + Vercel Postgres
- Vercel Blob
